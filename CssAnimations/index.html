Pra que aprender Animações?

Animações bem feitas, prendem a atenção do usuario, e melhoram a experiência em seu site

Feedback Interativo

Forneça feedback visual imediato sobre as ações do usuario com transições suaves

Narrativa Visual

Sites Modernos

Acompanha as tendências atuais de web design com efeitos visuais impressionantes

Tipos de Animações

Fade In/Out

Transições suaves de opacidade para entradas e saídas elegantes

.fade {
        animation: FadeIn 1s ease-in infinite;
}
    
@keyframes fadeIn {
    0% {
    opacity: 0;
    }
    
    50% {
    opacity: 1;
    }
    
    100% {
    opacity: 0;   
    }                      
}

Slide

Movimentação de elementos de forma controlada

.fade {
    animation: slideIn 1s ease-out infinite;
}
    
@keyframes slideIn {
    0% {
    transform: translateX(-100%);
    }
    
    50% {
    transform: translateX(0);
    }
    
    100% {
    transform: translateX(-100%);  
    }                    
}

Scale

Alterações de tamanhos para destacar elementos importantes.

.fade {
    animation: scale 2s infinite infinite;
}
    
@keyframes scale {
    0% {
         transform: scale(1); 
    }
    50% { 
        transform: scale(1.5); 
    }
    100% { 
        transform: scale(1); 
    }               
}
